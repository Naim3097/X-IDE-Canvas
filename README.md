# X.IDE

A powerful design tool built with React, Vite, and Konva.

## Features

-   **Design Editor**: A flexible canvas for creating designs (powered by tldraw).
-   **Social Media Templates**: Pre-made templates for various platforms.
-   **Content Planner**: Schedule your posts.
-   **Dashboard**: Central hub for all your creative work.

## Getting Started

1.  Install dependencies:
    \\\ash
    npm install
    \\\`n
2.  Start the development server:
    \\\ash
    npm run dev
    \\\`n
3.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

-   React
-   Vite
-   TypeScript
-   tldraw
-   Tailwind CSS
-   React Router DOM
